# Exercice 3

La fonction doit renvoyer l'objet qui a l'id 2

## Exemples

- Pour le tableau :

```javascript
[
  { id: 1, type: "house" },
  { id: 2, type: "house" },
  { id: 3, type: "destination" },
  { id: 4, type: "house" },
  { id: 5, type: "destination" },
];
```

- La fonction doit renvoyer :

```javascript
  {
    id: 2,
    type: "house"
  }
```
