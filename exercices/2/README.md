# Exercice 2

La fonction doit renvoyer un array de `house`

## Exemples

- Pour le tableau :****

```javascript
[
  { id: 1, type: "house" },
  { id: 2, type: "house" },
  { id: 3, type: "destination" },
  { id: 4, type: "house" },
  { id: 5, type: "destination" },
];
```

- La fonction doit renvoyer :

```javascript
[
  { id: 1, type: "house" },
  { id: 2, type: "house" },
  { id: 4, type: "house" },
];
```
